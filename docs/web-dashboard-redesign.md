# Web dashboard redesign — align with refined internal design tree model

## Overview

The web dashboard was built before the design tree, opsx-core FSM, and sidebar overhaul. Now that the internal model is more refined — proper tree hierarchy, lifecycle state machine, OpenSpec criteria — the web dashboard should reflect this. Full interactive tree browsing, node detail panels, OpenSpec file viewing (the thing the TUI defers to via OSC8), timeline/audit trail visualization. This is where deep browsing lives — the TUI sidebar is the at-a-glance view.
