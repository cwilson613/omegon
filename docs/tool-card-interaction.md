# Tool card selection, hover, and expand — keyboard + mouse navigation

## Overview

Tool cards in the conversation need proper selection/navigation: mouse hover highlights a card, arrow keys move between cards, Tab expands the focused card, Enter opens file links via OSC8. Currently Tab expand exists but there's no visual focus indicator and no way to navigate between cards. OSC8 hyperlinks for file paths in tool output (clickable in supporting terminals).
